# final-backend

โปรเจคนี้ประกอบด้วย 2 ส่วนหลัก:
- **client**: React Frontend
- **backend**: Express Backend API

## โครงสร้างโปรเจค
```
index.js
package.json
backend/
  package.json
  server.js
  .env
client/
  package.json
  README.md
  .env
  public/
  src/
```

## วิธีการใช้งาน

### 1. ติดตั้ง dependencies
```bash
cd backend
npm install
cd ../client
npm install
```

### 2. สร้างไฟล์ .env
- ดูตัวอย่างไฟล์ใน `client/.env` และ `backend/.env`
- ตั้งค่า Environment Variables ให้เหมาะสมกับการใช้งานและการ deploy

### 3. รัน Backend
```bash
cd backend
npm start
```

### 4. รัน Frontend
```bash
cd client
npm start
```

### 5. Deploy
- ดูรายละเอียดการ deploy ในไฟล์ `DEPLOYMENT.md`
